HTML Cheat Sheet

Forms Formatting and Attributes
Form Name Explanation
<form>...</form> Form

The Form element creates a form, spelling out how the form will operate based on it's Attributes
<label>...</label> Label

A simple label field telling the user what to enter in each field
<input/> Input The Form input attribute defines the type of field information to receive from a user

Input Attributes
Attribute Explanation
Type=""

Specifies the input type
name="" describes the name of the form
required Element must be completely filed out prior to submission
